# **App Name**: Axiom Trade Table Clone

## Core Features:

- Token Data Fetching: Fetch token data, including price, and new pairs. Data is mock.
- Real-time Price Updates: Implement real-time price updates using a WebSocket mock with smooth color transitions to indicate price changes.
- Token Table Display: Display token data in a table format, including columns for new pairs, final stretch, and migrated tokens.
- Interactive Elements: Implement interactive elements such as popovers, tooltips, and modals for additional information or actions.
- Sorting Functionality: Implement sorting functionality for all columns in the token table.
- Loading States: Implement loading states such as skeleton, shimmer, progressive loading, and error boundaries.
- Hover and Click Interactions: Implement hover effects and click actions on table rows and cells.

## Style Guidelines:

- Primary color: #1A202C, a dark shade to establish a professional look suitable for a financial product.
- Background color: #F7FAFC, a very light grey providing good contrast and a clean base for data presentation.
- Accent color: #4299E1, a vibrant blue used sparingly to draw attention to key interactive elements.
- Body and headline font: 'Inter', a sans-serif font, providing a modern, machined, objective, neutral look; suitable for both headlines and body text.
- Simple, geometric icons to represent token types and status.
- Responsive grid layout with clear content divisions, scaling down to 320px width.
- Subtle transitions and animations for price updates and interactive elements, maintaining a smooth user experience.
